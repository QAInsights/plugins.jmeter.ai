# PerfAtlas

A modern, high performance directory for JMeter plugins. 

PerfAtlas makes it easy to discover, search, and track the velocity of the best load testing extensions available. The directory features interactive historical usage charts and highlights AI ready tools for the modern testing ecosystem.

## Data
Automatically synced nightly from canonical JMeter plugin repositories via GitHub Actions 

## Development Commands

All commands are run from the root of the project.

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm run dev`             | Starts local dev server and fetches fresh data   |
| `npm run build`           | Builds the production static site                |
| `npm run fetch-plugins`   | Manually synchronizes the latest plugin datasets |
